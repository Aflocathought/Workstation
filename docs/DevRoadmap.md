### 0.1

### 0.2 + 0.3

- [x] 实现了Timeline功能，按时间条（Timeline）显示一整天的时间在应用的分布，以及其“每小时柱状图版（Timeline-HourlyBar）”，每个应用有对应的假随机颜色，现在应用层面的颜色比较丑，考虑后期优化^[FUTURE FEATURE]。

实现了“类目管理器”，主要功能是：分类有颜色，对应时间条上的颜色，能对分类进行创/删/改，有简单的导出类目配置。

并且为Timeline添加了按分类来查看时间的方案。点击某个色块会对应高亮所有同个应用。
[BUG #ff0] 在按类目绘制Timeline时，点击某个色块也会对应高亮那个应用。但是这不符合逻辑，实际上应该是高亮类目而非应用
[FEATURE]：一整天的时间Timeline剔除了不足10s的应用，但每小时柱状图是完全显示的，这是为了不让过多元素挤占空间影响Timeline比例（暂时无法解决这个问题）。
