# 快捷键提醒工具 (Shortcut Reminder)

## 功能概述

快捷键提醒工具是一个帮助用户记忆和管理应用程序快捷键的实用工具。它的主要特点包括：

- 🎯 **悬浮提示**：当按下修饰键（Ctrl、Alt、Shift）时，在窗口右下角显示可用的快捷键列表
- 📋 **分类管理**：按修饰键组合分组管理快捷键（Ctrl、Alt、Shift 及其组合）
- ✏️ **自定义配置**：支持添加、编辑、删除自定义快捷键
- 💾 **导出/导入**：完整的配置导出和导入机制
- 🔄 **持久化存储**：配置自动保存到本地存储

## 使用方法

### 1. 访问工具

在应用中导航到 **工具** → **快捷键提醒**

### 2. 查看快捷键

- 左侧边栏显示所有修饰键分组（Ctrl、Alt、Shift 等）
- 点击不同的分组可以查看该分组下的快捷键列表
- 每个快捷键显示组合键和功能描述

### 3. 悬浮提示

- 在应用的任何位置按下修饰键（如 Ctrl）
- 右下角会自动显示该修饰键下所有可用的快捷键
- 释放修饰键后提示框自动消失
- 支持组合键检测（如同时按下 Ctrl + Alt）

### 4. 添加快捷键

1. 点击顶部工具栏的 **➕ 添加快捷键** 按钮
2. 在表单中填写：
   - **按键**：输入按键名称（如 `A`、`F1`、`Enter`、`Space` 等）
   - **功能描述**：描述该快捷键的功能
3. 点击 **添加** 按钮保存

### 5. 编辑快捷键

1. 在快捷键列表中找到要编辑的项
2. 点击右侧的 ✏️ 编辑按钮
3. 修改信息后点击 **保存**

### 6. 删除快捷键

1. 在快捷键列表中找到要删除的项
2. 点击右侧的 🗑️ 删除按钮

### 7. 导出配置

1. 点击顶部工具栏的 **💾 导出配置** 按钮
2. 配置文件会自动下载为 JSON 格式
3. 文件名格式：`shortcut-config-{时间戳}.json`

### 8. 导入配置

1. 点击顶部工具栏的 **📁 导入配置** 按钮
2. 选择之前导出的 JSON 配置文件
3. 配置会自动导入并覆盖当前配置

### 9. 重置为默认

1. 点击顶部工具栏的 **🔄 重置默认** 按钮
2. 确认后会恢复到系统默认配置
3. ⚠️ 此操作会清除所有自定义快捷键

## 修饰键说明

工具支持以下修饰键和组合：

- **Ctrl**：单独的 Ctrl 键
- **Alt**：单独的 Alt 键
- **Shift**：单独的 Shift 键
- **Ctrl + Alt**：Ctrl 和 Alt 组合
- **Ctrl + Shift**：Ctrl 和 Shift 组合
- **Alt + Shift**：Alt 和 Shift 组合
- **Ctrl + Alt + Shift**：三键组合

## 默认快捷键

工具预置了一些常用的系统快捷键：

### Ctrl
- `Ctrl + S`：保存文件
- `Ctrl + C`：复制
- `Ctrl + V`：粘贴
- `Ctrl + X`：剪切
- `Ctrl + Z`：撤销
- `Ctrl + Y`：重做

### Alt
- `Alt + F4`：关闭窗口
- `Alt + Tab`：切换窗口

### Shift
- `Shift + Delete`：永久删除

### Ctrl + Alt
- `Ctrl + Alt + Delete`：任务管理器

### Ctrl + Shift
- `Ctrl + Shift + Esc`：任务管理器
- `Ctrl + Shift + N`：新建无痕窗口

## 配置文件格式

导出的配置文件采用 JSON 格式，结构如下：

```json
{
  "version": "1.0.0",
  "timestamp": 1699999999999,
  "config": {
    "ctrl": [
      {
        "key": "S",
        "description": "保存文件",
        "modifier": "ctrl"
      }
    ],
    "alt": [],
    "shift": [],
    "ctrl+alt": [],
    "ctrl+shift": [],
    "alt+shift": [],
    "ctrl+alt+shift": []
  }
}
```

## 技术实现

### 数据存储

- 使用 `localStorage` 进行本地持久化存储
- 存储键：`shortcut-reminder-config`
- 包含版本号和时间戳信息

### 键盘监听

- 监听全局 `keydown` 和 `keyup` 事件
- 实时检测修饰键状态（Ctrl、Alt、Shift）
- 根据修饰键组合显示对应的快捷键列表

### UI 设计

- 采用 VSCode 风格的界面设计
- 悬浮提示框使用半透明背景和模糊效果
- 支持平滑的淡入淡出动画

## 注意事项

1. **按键名称规范**：
   - 普通字母：`A`、`B`、`C` 等（大写）
   - 功能键：`F1`、`F2` 等
   - 特殊键：`Enter`、`Space`、`Escape`、`Tab`、`Delete` 等

2. **配置备份**：
   - 建议定期导出配置文件进行备份
   - 导入配置会完全覆盖当前配置

3. **浏览器兼容性**：
   - 工具依赖现代浏览器的 localStorage API
   - 建议使用最新版本的浏览器
