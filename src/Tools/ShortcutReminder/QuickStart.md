# 快捷键提醒工具 - 快速开始

## 快速访问

1. 启动应用
2. 导航到 **工具** → **快捷键提醒** ⌨️

## 5分钟快速上手

### 第一步：查看现有快捷键

- 左侧边栏显示 7 种修饰键分组
- 点击任意分组查看对应的快捷键列表
- 每个快捷键显示：组合键 + 功能描述

### 第二步：体验悬浮提示

1. 在应用的任何位置按下 **Ctrl** 键
2. 观察右下角出现的悬浮提示框
3. 尝试其他修饰键：**Alt**、**Shift**
4. 尝试组合键：**Ctrl + Alt**、**Ctrl + Shift** 等
5. 释放所有按键，提示框自动消失

### 第三步：添加自定义快捷键

1. 点击顶部的 **➕ 添加快捷键** 按钮
2. 填写表单：
   - **按键**: 输入 `F` (或其他按键)
   - **功能描述**: 输入 `搜索`
3. 点击 **添加** 按钮
4. 新快捷键出现在当前分组中

### 第四步：测试你的快捷键

1. 按下对应的修饰键（如 **Ctrl**）
2. 在右下角的提示框中看到你刚添加的快捷键
3. 释放按键

### 第五步：备份你的配置

1. 点击 **💾 导出配置** 按钮
2. 配置文件自动下载到本地
3. 妥善保管这个 JSON 文件

## 常用操作

### 编辑快捷键
- 点击快捷键右侧的 ✏️ 按钮
- 修改信息后保存

### 删除快捷键
- 点击快捷键右侧的 🗑️ 按钮

### 导入配置
- 点击 **📁 导入配置**
- 选择之前导出的 JSON 文件

### 重置配置
- 点击 **🔄 重置默认**
- 确认后恢复到初始状态

## 按键命名规范

| 按键类型 | 示例 |
|---------|------|
| 字母键 | `A`, `B`, `C` (大写) |
| 数字键 | `1`, `2`, `3` |
| 功能键 | `F1`, `F2`, `F12` |
| 特殊键 | `Enter`, `Space`, `Escape`, `Tab`, `Delete` |
| 方向键 | `ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight` |

## 修饰键组合说明

| 组合 | 说明 | 示例 |
|------|------|------|
| **Ctrl** | 单独 Ctrl 键 | `Ctrl + S` = 保存 |
| **Alt** | 单独 Alt 键 | `Alt + F4` = 关闭窗口 |
| **Shift** | 单独 Shift 键 | `Shift + Delete` = 永久删除 |
| **Ctrl + Alt** | 双键组合 | `Ctrl + Alt + Delete` |
| **Ctrl + Shift** | 双键组合 | `Ctrl + Shift + N` |
| **Alt + Shift** | 双键组合 | (自定义) |
| **Ctrl + Alt + Shift** | 三键组合 | (自定义) |

## 技巧提示

💡 **提示 1**: 配置会自动保存到浏览器本地存储，无需手动保存。

💡 **提示 2**: 定期导出配置文件作为备份，防止数据丢失。

💡 **提示 3**: 可以为不同的应用场景创建多个配置文件。

💡 **提示 4**: 在添加快捷键时，建议写清楚功能描述，方便日后查看。

💡 **提示 5**: 悬浮提示框不会影响其他操作，它是无交互的。

## 常见问题

**Q: 悬浮提示框不显示？**
A: 检查是否按下了修饰键，并且该修饰键下有配置的快捷键。

**Q: 配置丢失了怎么办？**
A: 如果有导出的配置文件，可以通过导入功能恢复。否则需要重新配置。

**Q: 可以为不同应用设置不同的快捷键吗？**
A: 当前版本暂不支持，这是后续开发计划中的功能。

**Q: 导入配置失败？**
A: 确保导入的是通过本工具导出的 JSON 文件，并且文件格式正确。

**Q: 快捷键冲突怎么办？**
A: 当前版本不检测冲突，请用户自行注意避免重复。
